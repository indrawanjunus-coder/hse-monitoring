Any authenticated user can submit POST /api/incidents with {"reporterId": 42} to attribute the incident to user 42 — across companies, at that. H9Auto-provisioned company admin password uses Math.random() [A02 / CWE-338] NEW (v3)
File

Evidence
function generatePassword(length = 10): string {
  const chars = "abcdefghjkmnpqrstuvwxyz...";
  let pw = "";
  for (let i = 0; i < length; i++) pw += chars[Math.floor(Math.random() * chars.length)];
  return pw;
}
Risk
Math.random() is a non-cryptographic PRNG (V8 xorshift128+). An attacker who observes enough output — or knows the approximate time of activation — can recover the PRNG state and predict future outputs. Fix
import { randomBytes } from "node:crypto";
// 10 chars base32 ≈ 50 bits entropy
return randomBytes(8).toString("base64url").slice(0, length);
